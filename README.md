# Contact-List

## Зависимости

На компьютере должена быть установлена LTS-версия [Node.js](https://nodejs.org/en/).

## Перед началом работы

Один раз нужно установить все зависимости.

### Серверная часть

```shell
cd server

npm i
```

### Клиентская часть

```shell
cd client

npm i
```

## Разработка

Запустить режим разработки.

### Серверная часть

```shell
cd server

npm run dev
```

### Клиентская часть

```shell
cd client

npm start
```
